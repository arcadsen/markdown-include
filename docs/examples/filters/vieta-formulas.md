下面是韦达定理（Vieta's Formulas）的[描述](#vieta)和证明。

::: {.theorem #vieta}

如果 $x_1,x_2$ 是一元二次方程 $ax^2 + bx + c = 0$ 的两个根，则有：

$$ x_1 + x_2 = -\dfrac{b}{a} $$

$$ x_1 \cdot x_2 = \dfrac{c}{a} $$

:::

::: proof

因为 $x_1,x_2$ 是一元二次方程 $ax^2 + bx + c = 0$ 的两个根，于是有：

$$ ax^2 + bx + c = a(x-x_1)(x-x_2) $$

根据乘法原理展开右式，比较等号两边的各项系数可得韦达定理的叙述。

:::
