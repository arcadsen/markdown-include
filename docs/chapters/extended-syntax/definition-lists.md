### 定义列表 Definition Lists

某些 Markdown 解析器允许你创建术语（terms）及其相应的定义（definitions）的列表，即 定义列表（definition lists）。要创建定义列表，请在第一行键入术语，然后在下一行中键入冒号，冒号后跟着空格和此术语的具体定义。
